# Pulse v8.5.1 BETA

Changes:
- Removed the theme switching button from the Home screen.
- Pulse is dark-only again.
- Archived chats can be opened from the Archived section.
- Select an archived chat using the normal Select mode or long-press.
- When all selected chats are archived, the toolbar shows Unarchive instead of Archive.
- Tap Unarchive to move the selected chats back to the normal Chats list.
- Existing chats and local preferences remain.
